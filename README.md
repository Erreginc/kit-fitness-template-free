## 🚀 How to Use the Free Version

This repository provides you with the high-converting **Hero Section** and the **Vectored Services Grid**.

### Step-by-Step Instructions:
1. Open the file **`kit-fitness-template.txt`** located right above in this repository.
2. Copy the entire block of raw HTML code.
3. Log into your **WordPress Dashboard**.
4. Create a **New Page** or open an existing draft.
5. Click the **`+`** icon (Add Block) in the Gutenberg editor and search for **"Custom HTML"**.
6. Paste the copied code directly into the block.
7. Click **Preview** or **Publish** to see the template instantly render on your site!
